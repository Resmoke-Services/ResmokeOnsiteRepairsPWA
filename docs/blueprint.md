# **App Name**: Resmoke Onsite PWA

## Core Features:

- Google Authentication: Enable users to securely log in using their Google accounts.
- Create Job: Allow users to create new job entries and store them in Firestore.
- Create Item: Enable users to add inventory items and save them to Firestore.
- Upload Image: Implement image uploading to Cloud Storage for associating with jobs or items.
- Load Job: Allow users to retrieve job details from Firestore.
- Load Item: Enable users to retrieve item/inventory information from Firestore.
- Load Image: Enable users to retrieve images from Cloud Storage.

## Style Guidelines:

- Primary color: Deep Blue (#293B5F) to convey trust and professionalism.
- Background color: Light Gray (#E7E5E4), provides a neutral backdrop to make the primary color stand out.
- Accent color: Teal (#3C8DAD) for interactive elements, such as buttons.
- Font pairing: 'Inter' (sans-serif) for body text and 'Space Grotesk' (sans-serif) for headings. Space Grotesk is suitable for headlines, while Inter will serve as body text.
- Use professional, consistent icons for actions like 'Create', 'Load', and 'Upload'.
- Maintain a clean, straightforward layout with clear navigation buttons. Keep forms concise.
- Use subtle animations for loading and transitions between pages to enhance user experience.
